[+]== WHO WE ARE ============================================================[+]
|                                                                              |
| We are a student led club who focus on the hands-on practical skills of      |
| ethical hacking and developing the skills needed to get started in           |
| cyber security. We welcome anyone and everyone regardless of skill.          |
| We aim to deliver a wide range of topics in the infosec world including      |
| forensics, web application testing, binary exploitation and more.            |
|                                                                              |
[+]==========================================================================[+]


[+]== WHAT WE DO ============================================================[+]
|                                                                              |
| At DISC we offer workshops, talks, competitions and more!                    |
| We host weekly meetups on the Burwood campus at 6pm in room B3.05            |
|                                                                              |
